How To Download Them
  • If your minceraft is running,
    1. Click a link above.
    2. Check your Download directory. (Windows: File Explorer, MacOS: Files)
    3. Open up minecraft, go to Options, and Resource Packs.
    4. Click Open resource pack folder.
    5. Drag and drop it from Download directory to resourcepack directory.
  • If your minecraft is NOT running,
    1. Click a link above.
    2. Check your Download directory. (Windows: File Explorer, MacOS: Files)
    3. Open up another file system and go ahead into .minecraft directory.
    4. Go ahead into resourcepack directory.
    5. Drag and drop it from Download directory to resourcepack directory.
